Мы, Старостин Вячеслав и Марк Панько, решили выполнить проект 'Создание игры "The Binding of Isaac" на pygame. Проект должен представлять из себя приложение при запуске которого будет открываться главная страница с выбором персонажа, выбором уровня сложности, выбора языка (между русским и английским) и с описанием игры. После выбора всех параметров игры будет предложено играть и игра будет запускаться. Сама игра будет подразумевать 2D-шутер с приключенческим сюжетом. На уровне будет несколько комнат с различными врагами. В игре можно собирать предметы: ключи, монеты, бомбы... Чтобы покинуть уровень нужно будет сразиться с боссом в комнате босса. Игрок может создавать несколько сохранений каждое из которых он может загрузить чтоюбы продолжить игру дальше. Проект будет реализован с помощью модуля pygame на языке Python.
